# Running for dev env
## Installing
1. Download [Git](https://git-scm.com/) and [Node](https://nodejs.org/)
2. `git clone https://github.com/abUwUser/beyond-software.git`
3. `cd beyond-software`
4. `npm i --global pnpm`
4. `pnpm i`

## Developing
Run `pnpm dev` in the project file to run the dev enviorment.

## Production
1. `pnpm build`
2. `pnpm preview`
